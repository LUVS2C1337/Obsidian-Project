## 核心心法

### 1. 工具各司其职，不要混用

❌ 用飞书记笔记 → 碎片化，无法关联
❌ 用 Obsidian 管任务 → 太重，不适合执行追踪
✅ 每个工具只做它最擅长的事

### 2. 信息的流动方向

```
新信息 → Claude(理解转化) → Obsidian(归档关联) → 飞书(排入计划) → 复习执行
                                              → NotebookLM(深度研读材料)
```

### 3. 最关键的一步：让信息流过你

大多数人收藏了就以为学会了。正确做法：

1. **遇到新知识** → 不直接保存
2. **跟 Claude 对话** → 用自己的话复述出来
3. **Claude 纠正理解偏差** → 确认真的懂了
4. **用自己的语言写入 Obsidian** → 建立关联
5. **飞书安排复习** → 间隔重复

## 四个层次的学习

### L1: 获取（暴露）
- 读书、听课、看视频
- 工具：飞书记录「学了什么」
- 度量：花了多少时间

### L2: 理解（内化）← 大多数人停在这
- 能用自己的话讲清楚
- 工具：**Claude** 对话、费曼练习
- 度量：能答对 Claude 的追问

### L3: 关联（网络化）
- 新知识和旧知识产生连接
- 工具：**Obsidian** 的 `[[双向链接]]`
- 度量：这个笔记被多少其他笔记引用了

### L4: 应用（外化）
- 解决问题、创造作品、教给别人
- 工具：飞书追踪产出
- 度量：产出了什么可交付的东西

## 每日学习 SOP

### 早上（5 分钟）
1. 飞书查看今日学习任务
2. 确认优先级

### 学习时段（25-50 分钟 / Pomodoro）
1. 输入材料（书/视频/文档）
2. **立刻打开 Claude**，做费曼练习：
   > 「我刚才学了 X，让我用自己的话讲一遍，你帮我检查理解对不对」
3. Claude 指出理解偏差 → 追问直到厘清
4. Claude 帮你提炼成结构化笔记 → **说「保存到 Obsidian」**

### 学习后（5 分钟）
1. Obsidian 中给笔记加上 `[[链接]]`
2. 飞书标记该任务完成
3. 如果学了有价值的东西，记一行收获

### 每周日晚（30 分钟）
1. Claude 回顾本周 Obsidian 学习日志
2. Claude 生成周报
3. 贴到飞书周报文档
4. 调整下周计划
