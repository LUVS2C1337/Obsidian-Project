## 冲突管理 (考试高频)

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| **合作/解决问题** | 找到根本原因，双赢 | 复杂问题，时间允许 |
| **妥协** | 各退一步 | 双方实力相当 |
| **缓和/包容** | 求同存异 | 关系 > 结果 |
| **强迫/命令** | 一言堂 | 紧急情况 |
| **撤退/回避** | 暂时搁置 | 需要冷静期 |

> **考试原则：** 合作/解决问题 是最优选，撤退 是最差选

## 团队发展五阶段

```
形成 → 震荡 → 规范 → 成熟 → 解散
```

| 阶段 | 特征 | 项目经理角色 |
|------|------|-------------|
| 形成 | 互相试探 | 指导 (Directing) |
| 震荡 | 冲突浮现 | 教练 (Coaching) |
| 规范 | 建立规则 | 支持 (Supporting) |
| 成熟 | 高效运转 | 授权 (Delegating) |
| 解散 | 收尾散伙 | 关怀 |
