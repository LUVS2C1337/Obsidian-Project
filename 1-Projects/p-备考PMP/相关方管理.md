## 相关方分析工具

### 权力/利益网格

```
        高
     ┌──────┬──────┐
 权  │ 令其 │ 重点 │
 力  │ 满意 │ 管理 │
     ├──────┼──────┤
     │ 最低 │ 随时 │
     │ 关注 │ 告知 │
     └──────┴──────┘
        低          高
          利益
```

### 参与度评估

| 级别 | 说明 |
|------|------|
| **不知晓** | 不知道项目存在 |
| **抵制** | 知道但反对 |
| **中立** | 不反对也不支持 |
| **支持** | 支持项目 |
| **领导** | 积极参与推动 |

> 目标：将相关方从不支持方向右移动
