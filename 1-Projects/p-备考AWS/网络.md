# 🌐 网络

## VPC（虚拟私有云）

| 概念 | 要点 |
|------|------|
| **CIDR** | VPC 用 /16 ~ /28，子网最小 /28 |
| **子网** | 公有子网 (有 IGW) · 私有子网 (无 IGW) |
| **IGW** | Internet Gateway — 公网访问入口 |
| **NAT Gateway** | 私有子网访问互联网 (单向) |
| **NACL** | 无状态子网防火墙 · 允许/拒绝规则 · 规则编号执行 |
| **Security Group** | 有状态实例防火墙 · 仅允许规则 · 默认拒入全出 |
| **VPC Peering** | 直连两个 VPC (不穿越公网) · 无传递性 |
| **Transit Gateway** | 星型拓扑连接多个 VPC/本地 · 有传递性 |
| **VPC Endpoint** | 私网访问 AWS 服务 (Gateway: S3/DynamoDB · Interface: 其他) |
| **Flow Logs** | 捕获 VPC 网络流量日志 → CloudWatch/S3 |

### NACL vs SG
| | NACL | Security Group |
|--|------|---------------|
| 层级 | 子网 | 实例/ENI |
| 状态 | **无状态** — 需分别配入/出站 | **有状态** — 出站自动允许响应流量 |
| 规则 | 编号执行，支持允许+拒绝 | 仅允许，全部规则评估 |
| 关联 | 1 子网 1 NACL | 多 SG 关联 1 实例 |

### VPC 端点对比

| 类型 | 服务 | 方式 |
|------|------|------|
| **Gateway Endpoint** | S3, DynamoDB | 路由表添加目标 |
| **Interface Endpoint** | 其他 AWS 服务 (APIGW, CW, SNS...) | ENI + DNS 解析 |

## Route 53（DNS 服务）

| 概念 | 要点 |
|------|------|
| **路由策略** | 简单 · 加权 · 延迟 · 故障切换 · 地理位置 · 地理邻近 · 多值回答 |
| **健康检查** | 关联 DNS 记录，自动故障切换 |
| **Alias 记录** | AWS 资源映射 (ELB/CloudFront/S3) — 免费，支持根域名 |
| **私有托管区** | VPC 内部 DNS 解析 |

## CloudFront（CDN）

| 概念 | 要点 |
|------|------|
| **起源 (Origin)** | S3 · ALB · EC2 · 自定义 · 媒体包 |
| **OAI/OAC** | 限制 S3 仅允许 CloudFront 访问 |
| **WAF** | CloudFront 可关联 WAF (Web ACL) |
| **Lambda@Edge** | 边缘节点执行代码 (修改请求/响应) |
| **Signed URL / Cookie** | 私有内容分发 |
| **地理限制** | 白名单/黑名单国家 |

### S3 跨区域复制 vs CloudFront
| | CRR | CloudFront |
|--|-----|------------|
| 延迟 | 跨区域直接 | 边缘节点缓存 |
| 动态内容 | 不支持 | 支持 |
| DDoS 防护 | 无 | 内置 (AWS Shield) |

## Direct Connect（专线）

| 概念 | 要点 |
|------|------|
| **带宽** | 50 Mbps ~ 100 Gbps |
| **延迟** | 稳定、一致 (比 VPN 好) |
| **VPN 备份** | Direct Connect + VPN (备用链路) |
| **DX Gateway** | 连通多个 VPC/区域 (需 Transit Gateway) |

> **VPN vs Direct Connect：** VPN = 便宜 公网加密 · DX = 贵 专线稳定
