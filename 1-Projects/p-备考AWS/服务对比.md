# 🔄 易混淆服务对比

---

## 存储

| 对比 | S3 | EBS | EFS | 实例存储 |
|------|-----|-----|-----|---------|
| 类型 | 对象存储 | 块存储 | 文件系统(NFS) | 块存储 |
| 持久性 | 11个9 | AZ 级 (快照备份) | AZ 级 | 实例终止丢失 |
| 访问 | HTTP API | 挂载到 1 个实例 | NFS 多实例 | 挂载到 1 个实例 |
| 扩展 | 无限 | 手动 (最大 64TB) | 自动 | 固定 |
| 跨 AZ | ✅ CRR | ❌ (可快照过去) | ✅ | ❌ |
| **场景** | 静态文件/备份 | OS/DB 盘 | 共享文件 | 临时缓存 |

## 数据库

| 对比 | RDS | Aurora | DynamoDB | Redshift |
|------|-----|--------|----------|----------|
| 类型 | 关系型 | 关系型优化 | NoSQL 键值/文档 | 数据仓库 |
| 扩展 | 垂直 (只读副本水平) | 自动存储 | 水平自动 | 节点扩容 |
| 延迟 | 毫秒 | 毫秒 | 毫秒/微秒(DAX) | 秒~分 |
| 场景 | OLTP · 复杂查询 | 高性能 OLTP · 兼容 MySQL/PG | 大规模键值 · 高并发 | 分析·报表·BI |

## 网络

| 对比 | VPC Peering | Transit Gateway | Direct Connect | VPN |
|------|-------------|----------------|---------------|-----|
| 拓扑 | 点对点 | 星型 Hub-Spoke | 专线直连 | 公网加密 |
| 传递性 | ❌ | ✅ | ✅ (TGW + DX) | ✅ |
| 带宽 | 取决 VPC | 高 | 50Mbps~100Gbps | 受限公网 |
| 成本 | 低 | 中 | 高 | 低 |
| **场景** | 少量 VPC 互通 | 大量 VPC/本地 | 关键业务稳定链路 | 备用/低成本 |

## 消息与事件

| 对比 | SQS | SNS | EventBridge | Step Functions |
|------|-----|-----|-------------|---------------|
| 模式 | 队列 (点对点) | 发布/订阅 | 事件总线 | 工作流编排 |
| 拉/推 | 拉 (Polling) | 推 (Push) | 推/拉 | 拉 (回调) |
| 消费 | 1 消费者 | 多订阅者 | 多目标 | 状态流转 |
| **场景** | 解耦/削峰 | 通知/告警 | 事件驱动架构 | 多步流程编排 |

## 安全

| 对比 | Security Group | NACL | WAF | Shield |
|------|---------------|------|-----|--------|
| 层级 | 实例/ENI | 子网 | 应用层 (L7) | 网络/应用 |
| 状态 | 有状态 | 无状态 | 有状态 | — |
| 规则 | 仅允许 | 允许+拒绝 | Web 攻击过滤 | DDoS 防护 |
| **场景** | 实例防火墙 | 网络ACL | SQL注入/XSS | DDoS |

## 计算

| 对比 | EC2 | Lambda | ECS Fargate | EKS |
|------|-----|--------|-------------|-----|
| 管理 | 全自行 | 无服务器 | 无服务器容器 | K8s 编排 |
| 运行 | 全天候 | ≤15min | 持久/短期 | 持久 |
| 扩展 | ASG | 自动 | 自动 | 手动/Cluster Autoscaler |
| 冷启动 | 无 | 有 | 无 | 无 |
| **场景** | 通用 | 事件驱动/短任务 | 容器部署 | 迁移已有 K8s |

## 集成服务

| 对比 | API Gateway | ALB | NLB |
|------|------------|-----|-----|
| 层级 | L7 | L7 | L4 |
| 协议 | HTTP/WebSocket | HTTP/HTTPS/WebSocket | TCP/UDP/TLS |
| 集成 | Lambda/HTTP/Mock | 目标组 (EC2/IP/Lambda) | 目标组 (EC2/IP/ALB) |
| 认证 | IAM/Cognito/Lambda Authorizer | OIDC (在 ALB 层面) | 无 |
| **场景** | REST/HTTP API · WebSocket | HTTP 负载均衡 | TCP/UDP 低延迟 |
