## 🔧 三工具协同工作流

```
飞书 (执行管理)          Claude (智能导师)         Obsidian (知识库)
─────────────────      ─────────────────      ─────────────────
每日任务 + 打卡 ──→                              
模考安排 + 提醒 ──→                              
                        ←── 错题分析 + 讲解      → 错题本永久存档
                        ←── 作文批改 + 范文      → 写作模板库
                        ←── 口语话题练习          → 口语语料库
                        ←── 词汇精讲              → 词汇网络图
                        ←── 语法答疑              → 语法知识库
                        ←── 阅读解题技巧          → 技巧兵器库
飞书文档 ←── Claude 生成的学习周报 ────────────────
飞书表格 ←── 模考成绩趋势分析 ─────────────────────
```

## 飞书侧设置

### 1. 任务清单（飞书多维表格）
- 字段：日期 | 模块 | 任务内容 | 时长 | 完成状态 | 备注
- 按周视图分组，追踪每日进度

### 2. 模考成绩追踪表
- 字段：日期 | 听力 | 阅读 | 写作 | 口语 | 总分 | 错题链接

### 3. 每日提醒机器人
- 早上 8:00 — 今日学习计划
- 晚上 22:00 — 提醒回顾 + 打卡

### 4. 资料库（飞书文档）
- 储存 PDF 真题、听力音频等大文件
- 定期归档 Claude 生成的周报
