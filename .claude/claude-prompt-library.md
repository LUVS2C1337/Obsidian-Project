# claude-prompt-library

> 经过验证的高效提示词集合

## 编程

### 代码审查
```
请审查以下代码的：安全性、性能、可维护性
```

### 重构建议
```
这段代码有什么可以改进的地方？按优先级列出
```

## 知识管理

### 笔记整理
```
将以下笔记重组为 PARA 结构，建议放入哪个目录
```

### 知识萃取
```
从以下对话中提取 3 个核心洞察，写成可归档的笔记
```

## 学习

### 费曼解释法
```
用简单的语言向一个完全不懂的人解释 [概念]
```

### 对比分析
```
对比 A 和 B 的：优缺点、适用场景、学习曲线
```
